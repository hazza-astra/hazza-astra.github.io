Emirates Mars Mission

Research Intern

A project focusing on spider formations in the cryptic region of Mars found in pockets of sublimated ice near the Martian South Pole. The scope of the project was transferring data structures as well as essential functions to process spatial data from MATLAB to Python to allow for use of data visualization through Bokeh. From which, a model was trained through TensorFlow to use spatial data for density-based clustering. Astral Realms

Rogue-like Video Game

An in progress, personal passion, rogue-like dungeon crawler 2D game influenced by the old style of 90s videogames with old school sprites coded in two different languages (GodotScript and Python.) The main focus of the project itself in building the data structures of each system and finding optimal structures for each system in the game.


Working between both GodotScript as well as Python proved an interesting challenge as it meant that I had to follow the same structures between languages and build classes such as the player in a way that they communicate without error. Moreover, the randomization is a key aspect of the game, from the philosophy of "how random is pseudorandom?" to convoluted algorithms to generate rooms and auto-check for errors along the way.
